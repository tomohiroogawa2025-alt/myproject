# Spectral ML Practice Workspace

スペクトル解析と機械学習を学びながら、自分用の実行手法を整えるための作業プロジェクトです。

## Structure

- `SpectralAnalysisPractice/`: 参考教材リポジトリ。Git 管理対象外。
- `my-notebooks/`: 自分で試す notebook。
- `scripts/`: 再利用する処理を Python スクリプトとして置く場所。
- `data/`: 自分の入力データ。Git 管理対象外。
- `outputs/`: 図、モデル、予測結果などの出力先。Git 管理対象外。
- `notes/`: 学習メモ、実験メモ、手順メモ。

## Current Direction

教材 notebook を読みながら、以下の流れを自分用に整理していきます。

1. データ読み込み
2. スペクトルの可視化
3. 前処理
4. 学習用、評価用データへの分割
5. モデル学習
6. 評価
7. 結果とモデルの保存
