# My Notebooks

教材を参考にしながら、自分のデータで試す notebook を置く場所です。

教材リポジトリ内の notebook は直接編集せず、必要な処理をここに写して検証します。

## Production notebooks

- `01_spectral_data_exploration_production.ipynb`: データ構造とスペクトルの確認
- `02_pls_operational_modeling_production.ipynb`: PLS単一モデルの運用版
- `03_advanced_model_selection_ensemble_production.ipynb`: 既存の高度モデル探索版
- `04_preprocess_model_interval_ensemble_production.ipynb`: 前処理探索、全波長モデル探索、波長区間選択、アンサンブルを順に整理した版
- `05_visual_chemometrics_modeling_production.ipynb`: モデル構築前のデータ可視化・ケモメトリクス診断専用版
- `06_species_kl_classification_production.ipynb`: 樹種分類とN/L材分類の可能性診断・分類モデル評価版
- `07_species_kl_classification_900_1700nm_production.ipynb`: 900-1700 nm相当のみを使った樹種分類とN/L材分類の可能性診断・分類モデル評価版
- `08_nl_mixture_ratio_estimation_production.ipynb`: N材/L材混合パルプの比率推定に向けた疑似混合スペクトル評価版
